# 🎨 My Style - Портфоліо Сайт

Сучасний, красивий портфоліо-сайт з гарною hero-обкладинкою, анімаціями та responsive дизайном.

## ✨ Особливості

- **Сучасний дизайн** - gradient обкладинки, плавні анімації
- **Hero Section** - вражаючий вступ з рухомими градієнт-орбами
- **Responsive** - відповідає на мобільних, планшетах та десктопі
- **Smooth Scroll** - плавна навігація між секціями
- **Animations** - листи, плаваючі карточки, ripple ефекти
- **Modern Stack** - Vanilla HTML, CSS, JavaScript (без залежностей)

## 🚀 Розпочати роботу

1. Клонуйте репозиторій:
```bash
git clone https://github.com/bogoslavsulga-debug/my-style.git
cd my-style
```

2. Відкрийте `index.html` у браузері або використовуйте Live Server:
```bash
# Якщо у вас встановлено Node.js
npx live-server
```

3. Налаштуйте файли під себе:
   - Змініть текст у `index.html`
   - Додайте свої кольори у `:root` в `styles.css`
   - Персоналізуйте `script.js` для своїх потреб

## 📁 Структура

```
my-style/
├── index.html      # HTML структура
├── styles.css      # Styling та анімації
├── script.js       # Інтерактивність
└── README.md       # Документація
```

## 🎨 Кольорова схема

Можна змінити в `:root` змінних у `styles.css`:

```css
:root {
    --primary: #6366f1;      /* Основний колір */
    --secondary: #8b5cf6;    /* Допоміжний */
    --accent: #ec4899;       /* Акцентний */
    --dark: #0f172a;         /* Темний фон */
    --light: #f8fafc;        /* Світлий фон */
}
```

## 📱 Секції

1. **Навігація** - фіксована, з blur ефектом
2. **Hero** - головна обкладинка з градієнтами та орбами
3. **Про мене** - інформація та скілли
4. **Роботи** - портфоліо проектів
5. **Контакти** - CTA секція
6. **Footer** - завершення

## 🔧 Налаштування

### Додавання нових проектів
Добавьте новий блок у `.work-grid`:

```html
<div class="work-card">
    <div class="work-image gradient-bg-1"></div>
    <h3>Ваш Проект</h3>
    <p>Описання</p>
    <a href="#" class="work-link">Дивіться →</a>
</div>
```

### Зміна градієнтів
Відредагуйте CSS змінні для своїх кольорів:

```css
--gradient-1: linear-gradient(135deg, #ваш-колір1 0%, #ваш-колір2 100%);
```

## 🌐 Розгортання

### GitHub Pages
1. Перейдіть на Settings репозиторію
2. Виберіть "Pages"
3. Виберіть Branch: `main` та Folder: `/ (root)`
4. Ваш сайт буде доступний за посиланням

### Інші хостинги
- **Vercel** - простий Push-to-Deploy
- **Netlify** - підтримує Git інтеграцію
- **Surge.sh** - швидкий деплой
